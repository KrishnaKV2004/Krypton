# 🌌 Krypton プログラミング言語

---

## ✨ イントロダクション

```
"静けさが光るコードの世界で、  
気高き言語は静かに流れる。  
星と光のスペクトルから生まれ、  
Krypton はバイナリの力で輝く。"
```

**Krypton (クリプトン)** は、現代的でエレガント、そして表現力が高いプログラミング言語です。元素の **クリプトン (原子番号36)** にちなんで名付けられ、その安定性、多用途性、および **電子配置** の構造をモデルにした設計思想を持ちます。

---

## 🧪 なぜ名前が「Krypton」なのか

この名前は美しいだけでなく、**科学的な意味** も持っています。

* **原子番号**: `36`
* **電子配置**:
  `1s² 2s² 2p⁶ 3s² 3p⁶ 3d¹⁰ 4s² 4p⁶`

電子層ごとに Krypton 言語の構成要素を表しています:

* **2 つのコアモジュール** → `1s²`
* **8 つのプライマリデータ型** → `2s² 2p⁶`
* **18 の組み込み関数** → `3s² 3p⁶ 3d¹⁰`
* **8 の標準ライブラリ** → `4s² 4p⁶`

```
"構造明確な Krypton の光の内、  
データが踊る、電子の舞い。"
```

## ⚙️ 2 つのコアモジュール

```
"根心にはたった2つのモジュール、  
それが言語の心と魅力。  
一つは言語を斬る理性、  
一つは記憶を管理する素早さ。"
```

* **`core.logic`** ― 分岐と制御のロジック
* **`core.memory`** ― 変数とメモリ管理

---

## 🔢 8 つのプライマリデータ型

```
"バイナリの息中で生まれた構造の実、  
のちの八つの定義が形を成す"
```

* `num` ― 数値 (int や float)
* `text` ― 文字列 (Unicode)
* `bool` ― 真偽値 (`true`, `false`)
* `list` ― 順序ありの集合
* `map` ― キーバリュー式のペア
* `void` ― 空の表示
* `func` ― 関数 (フーストクラス)
* `byte` ― 低級操作用の生のバイト
